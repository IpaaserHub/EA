## セクション3: 収益モデルの選択肢

### 3つの収益モデル比較

| モデル | 内容 | 登録 | 推奨度 |
|--------|------|------|--------|
| **売り切り型** | ¥50,000で販売、終わり | ❌ 不要 | ⭐⭐⭐ 推奨 |
| **成功報酬型** | 顧客の利益の20%をもらう | ✅ 必要 | ⭐ 高コスト |
| **アフィリエイト併用** | 売り切り + 証券会社紹介料 | ❌ 不要 | ⭐⭐ 検討可 |

### モデル1: 売り切り型（推奨）

```
顧客 → ¥49,800支払い → EA受け取り → 終了

✅ メリット:
・登録不要
・シンプルな取引
・法的リスク最小

❌ デメリット:
・継続収入なし
・1回きりの売上
```

### モデル2: 成功報酬型（現時点では非推奨）

```
顧客の利益 ¥100,000
    ↓
御社の取り分 ¥20,000（20%）

❌ 問題点:
・投資助言・代理業の登録が必要
・登録費用: ¥150,000
・営業保証金: ¥5,000,000
・顧客口座へのアクセスが必要
・継続的な関係性 = 規制対象
```

**金融庁の見解:**
> "成功報酬制を採用する投資助言・代理業者では、顧客の取引履歴の確認のために、閲覧権限のみ割り当てられた銀行・証券口座のID及びパスワードの割り当てを受けるケースが存在"

→ 利益の一部をもらう = **投資助言業として扱われる**

### モデル3: アフィリエイト併用（検討可）

```
┌─────────────────────────────────────────┐
│  売り切り + 証券会社アフィリエイト      │
│                                         │
│  1. EAを¥49,800で販売                   │
│  2. 「このEAはXX証券で使えます」        │
│  3. 紹介リンク経由で口座開設            │
│  4. 証券会社から紹介料をもらう          │
│                                         │
│  → 1人あたり ¥49,800 + ¥15,000 = ¥64,800│
└─────────────────────────────────────────┘
```

**アフィリエイト報酬の目安:**

| 紹介先 | 報酬目安 | リスク |
|--------|----------|--------|
| 国内FX業者（DMM FX等） | ¥5,000〜30,000/人 | ✅ 低い |
| 国内暗号資産取引所 | ¥1,000〜10,000/人 | ✅ 低い |
| 海外FX業者（XM等） | $50〜200/人 | ⚠️ グレーゾーン |

**注意: 海外業者のリスク**
> "金融庁は2025年時点で数百社を超える海外FX業者に警告を発出"

海外無登録業者への送客は、金融庁から警告される可能性があります。

### 推奨アプローチ

| 優先度 | モデル | 理由 |
|--------|--------|------|
| **1位** | 売り切り型のみ | 登録不要、シンプル、低リスク |
| **2位** | 売り切り + 国内証券アフィリ | 追加収益、比較的安全 |
| **3位** | 成功報酬型 | ¥5M+必要、将来の選択肢として |

## セクション5: 法的要件・リスク対策

### 売り切り型で登録不要にする条件

```
✅ 以下を全て満たせば投資助言業登録は不要

□ 会員登録なしで誰でも購入可能
□ 買い切り型（サブスク禁止）
□ 継続的なシグナル配信なし
□ 投資助言としてのサポートなし
□ ソフトウェア単体で動作（外部データ不要）
```

### 必須の法的表記

| 項目 | 内容 |
|------|------|
| **特定商取引法に基づく表記** | 会社名、住所、電話番号、代表者名 |
| **免責事項** | 投資助言ではない旨を明記 |
| **リスク説明** | 元本割れリスクの説明 |
| **禁止事項** | 「必ず儲かる」等の表現は違法 |

### 免責事項テンプレート

```
【重要事項・免責事項】

・本ソフトウェアは投資助言・代理業に該当するものではありません
・本ソフトウェアの利用による投資判断は全てお客様自身の責任で
  行っていただきます
・過去の運用実績は将来の利益を保証するものではありません
・FX取引・暗号資産取引には元本を上回る損失が発生するリスクが
  あります
・当社は本ソフトウェアの利用により生じた損害について
  一切の責任を負いません
```

### やってはいけないこと（違法リスク）

| ❌ 禁止 | 理由 |
|--------|------|
| 「月利10%保証」 | 断定的判断の提供（違法） |
| 「必ず儲かる」 | 誇大広告（違法） |
| 会員限定販売 | 投資助言業登録が必要になる |
| 月額課金 | 投資助言業登録が必要になる |
| シグナル配信 | 投資助言業登録が必要になる |

### リスク対策チェックリスト

```
✅ 売り切り型のみ
✅ 免責事項を必ず明記
✅ 実績は「過去データ」と明記
✅ 利用規約で責任範囲を限定
✅ 誇大広告を絶対にしない
✅ 購入前に十分なリスク説明
```
